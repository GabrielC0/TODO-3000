## ⚡ Installation Avancée (Optionnelle)

Si vous voulez utiliser les versions Python :

### Version Console
```bash
python todo_3000.py
```

### Version Web Server
```bash
# Installer Flask
pip install flask

# Lancer le serveur
python todo_3000_web.py

# Ouvrir dans le navigateur
# http://localhost:5000
```

## 🎨 Personnalisation

### Modifier les Messages IA
```javascript
// Dans le fichier HTML/JS
this.aiResponses = [
    "Votre message personnalisé...",
    "Autre réaction de l'IA...",
    "Ajoutez vos propres phrases !"
];
```

### Changer les Couleurs
```css
/* Couleurs principales */
:root {
    --primary-color: #00ffff;    /* Cyan */
    --secondary-color: #ff00ff;  /* Magenta */
    --accent-color: #00ff00;     /* Vert */
}
```

2. **Navigateur moderne** (Chrome, Firefox, Safari, Edge)
3. **Permissions localStorage** autorisées
4. **Console développeur** pour les erreurs (F12)
